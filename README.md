**A fast, elegant network speed tester for Windows**

ExitPing is a lightweight desktop application that provides real-time internet speed measurements right from your system tray. Get instant insights into your download/upload speeds and connection quality with a beautiful, minimalist interface.

## ✨ Features

- **Real-Time Speed Testing** — Measure download, upload, and ping speeds with precision
- **Automatic Server Selection** — Intelligently picks the best test server based on your location
- **Network Identity Detection** — Instantly displays your ISP, IP address, and geographic location
- **Minimal Design** — Compact, frameless window that stays out of your way
- **System Tray Integration** — Access from your system tray for quick testing
- **Auto-Testing** — Optional automatic speed tests at configurable intervals
- **Startup Shortcut** — Launch ExitPing automatically with Windows

## Installation

### Download & Install

1. Visit the [Releases](https://github.com/yourusername/exitping/releases) page
2. Download the latest **ExitPing Setup** installer or portable version
3. Run the installer and follow the setup wizard
4. Launch from your Start Menu or system tray

### System Requirements

- **OS:** Windows 10 or later (64-bit)
- **RAM:** 128 MB minimum
- **Internet Connection:** Required for speed testing

## Quick Start

1. **Launch ExitPing** — Click the app icon or run from Start Menu
2. **Click the Speed Test Button** — Begin testing your connection
3. **View Results** — Watch real-time metrics update as the test runs
4. **Check Details** — Tap the settings icon to view ISP, IP, and server information

## Settings

Access the settings panel to customize:

- **Enable at Startup** — Automatically launch ExitPing when Windows starts
- **Auto Test** — Schedule automatic speed tests at regular intervals
- **Server Selection** — Switch between geographic server locations

## Troubleshooting

**Tests not running?**
- Ensure you have an active internet connection
- Check your firewall settings
- Try closing and reopening the application

**Inaccurate results?**
- Disable other bandwidth-heavy applications
- Run the test multiple times for consistency
- Ensure you're not on a VPN for most accurate results

**Application won't start?**
- Restart your computer
- Reinstall the application
- Check Windows Defender or antivirus isn't blocking it

## Understanding Your Results

| Metric | Description |
|--------|-------------|
| **Download** | Speed at which data is received (Mbps) |
| **Upload** | Speed at which data is sent (Mbps) |
| **Ping** | Response time to the test server (ms) |
| **ISP** | Your Internet Service Provider name |
| **Server** | Geographic location of the test server |
